# Flask Projects

Welcome to the **Flask Projects** repository! This repository serves as a collection of simple, practical Flask applications designed to demonstrate core web development concepts in Python, ranging from basic UI rendering to user authentication and dynamic templating.

## 🚀 Projects Included

| Project Name | Description | Key Features |
| :--- | :--- | :--- |
| **Basic_flask_ui** | A fundamental starter project establishing a basic web user interface using Flask. | Routing, basic HTML rendering |
| **jinja2** | An exploration of the Jinja2 templating engine to dynamically inject backend data into front-end views. | Template inheritance, control structures (`if`, `for`), loops |
| **Login_System_Basic** | A lightweight implementation of a user authentication mechanism. | Form handling, basic conditional validation |
| **user_login_system** | A more comprehensive login system managing user access and session workflows. | Session management, secure routing, simulated user database |

---

## 🛠️ Tech Stack

* **Backend:** Python 3.x, Flask
* **Frontend:** HTML5, CSS3, Jinja2 Templating
